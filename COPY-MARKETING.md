**Pour qui ?**
- Voyageurs d'affaires (B2B principal)

2. Suivi de vol en temps réel — il vous attend même si votre avion est en retard
3. Mercedes exclusivement — pas de Dacia ou Peugeot déguisé
4. Bilingue — clientèle internationale à l'aise
5. Licence VTC officielle N°013230073 — confiance et légalité

## ✍️ RÈGLES DE COPYWRITING

### Les 5 principes
1. **Bénéfice avant caractéristique** → "Vous arrivez à l'heure" (bénéfice), pas "Suivi GPS en temps réel" (caractéristique)
2. **Concret avant abstrait** → "06 15 96 32 75" visible immédiatement, pas "Contactez-nous"
3. **Vous, pas Nous** → Parler au client, pas de soi. "Votre transfert" plutôt que "Notre service"
4. **Confiance par la preuve** → Licence VTC, nombre de clients, avis Google, marque Mercedes
5. **Urgence douce** → "Disponible maintenant" plutôt que "Réservez vite !!!"

### Ton
- Premium mais accessible — pas arrogant
- Rassurant — le client a peur de rater son avion
- Direct — phrases courtes, verbes d'action
- Chaleureux — Provence = soleil, accueil, art de vivre

## 📝 COPY PAR SECTION

### HERO
**Badge :**
`⭐ Service Premium 24h/24 — Licence VTC N°013230073`
→ Rassure immédiatement sur la légitimité

**H1 :**
```
Chauffeur Privé
Aix-en-Provence
```
→ Simple, direct, le mot-clé principal en premier

**Sous-titre :**
`Transferts aéroport, voyages d'affaires, événements.
Mercedes haut de gamme. Tarifs fixes. Toujours à l'heure.`
→ 3 arguments en 3 lignes, chaque mot compte

**CTA principal :**
`Réserver mon transfert →`
→ "mon" = personnalisé, "→" = action immédiate

**CTA secondaire :**
`Appeler maintenant`
→ "maintenant" = disponibilité immédiate

### ATOUTS
Ne pas écrire "Nos atouts" — trop générique. Titres directs :
- "Tarif fixe confirmé" (pas "Tarif Fixe")
- "Toujours à l'heure" (pas "Ponctualité")
- "Français & Anglais" (pas "Bilingue")
- "Assurance complète" (pas "Assuré")

### SERVICES
Label : `CE QUE NOUS FAISONS POUR VOUS`
H2 : `Un service adapté à chaque trajet`

Chaque service → formule "Vous + bénéfice" :
- Aéroport → "Vous atterrissez, on est là." (pas "Service transfert aéroport")
- Affaires → "Vos appels, votre espace, votre tranquillité."
- Événements → "Votre soirée commence dans la voiture."

CTA carte : `Réserver ce service →`

### FLOTTE
Label : `VOS VÉHICULES`
H2 : `Mercedes exclusivement. Parce que le détail compte.`

Chaque véhicule :
- S-Class : "L'excellence à l'état pur. Pour ceux qui n'acceptent pas le compromis."
- E-Class : "Performance et élégance. Votre bureau mobile."
- V-Class : "Tout le groupe ensemble. Zéro stress, zéro logistique."

### ZONES
Label : `OÙ NOUS INTERVENONS`
H2 : `Aix-en-Provence et toute la région`
Sous-titre : `Un seul appel, n'importe où dans la région PACA.`

### AVIS
Label : `ILS NOUS FONT CONFIANCE`
H2 : `Ce que nos clients disent`
→ Laisser les témoignages parler — pas de paraphrase

### FAQ
H2 : `Vos questions, nos réponses`
→ Réponses directes, une phrase max pour commencer avant le détail

### CONTACT
Label : `RÉSERVATION`
H2 : `Réservez votre transfert`
Sous-titre : `Réponse garantie sous 2h. Devis gratuit et sans engagement.`

Bouton submit : `Envoyer ma demande →`
Message de confirmation : `Demande reçue ! Nous vous rappelons dans les 2 heures.`

## 📞 CALLS-TO-ACTION — Priorité et placement

### Hiérarchie CTA
1. **Appel téléphonique** → conversion la plus rapide (taxi = urgence)
2. **Formulaire** → planification à l'avance
3. **WhatsApp** → préférence internationale

### Placement des CTAs
- Nav : numéro de téléphone visible en permanence
- Hero : "Réserver" + "Appeler maintenant" above the fold
- Après chaque section service : lien "Réserver ce service →"
- Sticky mobile : bouton "Appeler" fixe en bas d'écran
- Footer : numéro + email + WhatsApp
- FAQ : CTA de réservation après la dernière question

### Règle du CTA sticky mobile
```jsx
// Bouton flottant mobile uniquement
<a href="tel:+33615963275"
   className="fixed bottom-6 left-1/2 -translate-x-1/2 z-50
              bg-gold text-dark font-bold px-8 py-4 rounded-full
              shadow-lg md:hidden">
  📞 Appeler maintenant
</a>
```
