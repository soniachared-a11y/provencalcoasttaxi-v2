⚠️ Statut actuel : Le site est techniquement prêt (sitemap, robots.txt, schemas structurés). Il manque uniquement la vérification Google Search Console pour déclencher l'indexation. Temps estimé : 10 minutes.
🔧 Étapes pour indexer le site
1
🔴 Urgent
Ouvrir Google Search Console
Accès gratuit — compte Google requis

Rendez-vous sur :

👉 search.google.com/search-console

Connectez-vous avec le compte Google associé à votre activité. Cliquez sur "Ajouter une propriété".

2
🔴 Urgent
Ajouter le domaine
Choisir le type "Préfixe d'URL"

Dans la boîte de saisie, entrez :

https://taxisprovencaleaix.fr

Sélectionnez "Préfixe d'URL" (pas "Domaine") → Cliquez "Continuer".

3
🔴 Urgent
Vérifier la propriété
Via fichier HTML (méthode recommandée)

Google vous propose plusieurs méthodes. Choisissez "Fichier HTML".

  • Google vous donnera un fichier nommé : googleXXXXXXXXXXXXXXXX.html
  • Téléchargez ce fichier et placez-le dans le dossier /public/ du projet
  • Déployez le site (le fichier doit être accessible à l'URL indiquée par Google)
  • Cliquez "Vérifier" dans Search Console

Note : Le fichier de vérification PLACEHOLDER est déjà créé dans /public/ — il suffit de le remplacer par le vrai fichier Google.

4
🟠 Important
Soumettre le Sitemap
Déclenche l'exploration de toutes les pages

Une fois la propriété vérifiée :

  • Menu gauche → "Sitemaps"
  • Dans le champ, entrez :
sitemap.xml

Cliquez "Envoyer". Google va explorer et indexer les 6 pages du site.

Pages qui seront indexées :

  • ✓ taxisprovencaleaix.fr/ (Accueil)
  • ✓ taxisprovencaleaix.fr/services
  • ✓ taxisprovencaleaix.fr/flotte
  • ✓ taxisprovencaleaix.fr/a-propos
  • ✓ taxisprovencaleaix.fr/contact
  • ✓ taxisprovencaleaix.fr/mentions-legales
5
🟠 Important
Demander l'indexation manuelle
Pour accélérer le processus

Dans Search Console :

  • Menu gauche → "Inspection d'URL"
  • Entrez chaque URL une par une
  • Cliquez "Demander l'indexation" pour chacune

À faire pour ces URLs en priorité :

https://taxisprovencaleaix.fr/ https://taxisprovencaleaix.fr/services https://taxisprovencaleaix.fr/contact

Indispensable pour le Local Pack Google Maps
